# Focus OS V8.0 "Pro Developer" 🚀

Focus OS is an elite, multi-platform productivity and deep-focus application designed to merge gamification, AI task management, biological pacing, and now **professional-grade coding** into a singular, state-of-the-art environment.

## 🌟 The Epics of Focus OS

### Epic 1: Deep Analytics & Heatmaps
Stop guessing how productive you are. Focus OS visually maps your focus time across a 365-day GitHub-style heatmap, generating detailed statistics of your deep work habits.

### Epic 2: Live Multiplayer & Social Sync
Focusing alone is hard. Create encrypted study rooms with passwords. Admin users can accept/reject peers. Everyone in the room is synced: if someone loses focus, their avatar goes red for everyone to see.

### Epic 3: The Native & Web IDE Ecosystem
No more simple text boxes. Focus OS features an embedded Monaco Editor with syntax highlighting and a multi-file browser synced to the cloud.
- **WebContainers:** Run `npm install`, boot Node.js servers, and execute code entirely within your browser!
- **Polyglot Execution:** First-class support for C++ and Java via the Piston API.
- **Live Web Preview:** Running a React or Vite app? A live preview iframe automatically docks to the side of your editor the moment your dev server starts.
- **XTerm.js Terminal:** A fully interactive CLI terminal built straight into the UI.

### Epic 4: Real-Time Code Collaboration (Google Docs for Code)
Powered by **Yjs** and **WebRTC**, invite your team and watch their cursors move in real-time as you code together! No central server required for rapid peer-to-peer syncing.

### Epic 5: AI "Ghost Text" & Error Diagnostics
- **Ghost Text Auto-Complete:** Just like GitHub Copilot, pause your typing to see blazing-fast inline code completions generated by Gemini 2.5 Flash.
- **Explain Error Console:** If your terminal crashes, click the new `Explain Error` button to have Gemini instantly diagnose your stack trace and active code.

### Epic 6: Developer Tooling (Vim, Themes & Snippets)
- **Vim Keybindings:** Toggle the new Vim mode to get full Vim motions (`hjkl`) directly within Monaco.
- **Snippets Library:** A dedicated side panel to save your most used code blocks to the cloud and drop them instantly into your editor.
- **Custom Theme Engine:** Quickly switch between Dark, Light, and High Contrast editor themes from the top toolbar.

### Epic 7: Health & Biometric Protocols
A revolutionary "Heartbeat" core timer pulses dynamically to anchor you in flow. 
- **Spine Check:** Every 15 minutes, you are reminded to align your posture.
- **Hydration:** Every 30 minutes, you are reminded to drink water.
- **Vision Protection:** The strict 20-20-20 rule locks your screen every 20 minutes to prevent macular degradation.

### Epic 8: AI Kanban & Neural Tasking
A drag-and-drop workflow board right inside your focus session. Throw a massive goal at the "AI Breakdown" engine, and Gemini will instantly populate your board with actionable micro-tasks.

### Epic 9: Jarvis Voice AI & The Focus Pet
A virtual companion to hold you accountable. The Focus Pet sleeps peacefully while you work, but wakes up angry if you break your streak. Meanwhile, the integrated Gemini Voice Assistant will actually speak to you out loud, keeping you guided.

## 🛠️ Architecture & Setup
- **Web Build:** React + Vite
- **Desktop Build:** Electron
- **Browser Execution Engine:** WebContainers API
- **Code Editor:** Monaco Editor + XTerm.js
- **Collaboration Sync:** Yjs + y-webrtc + y-monaco
- **Database:** Firebase Firestore & Auth
- **AI Core:** Google Gemini 2.5 Flash

### Installation
1. `npm install`
2. `npm run dev` (Web) or `npm run electron:dev` (Desktop Native)
3. Add your Gemini API Key directly inside the app's Setup screen.
4. Add your Firebase config to `firebase.js` (Optional, runs locally otherwise).
